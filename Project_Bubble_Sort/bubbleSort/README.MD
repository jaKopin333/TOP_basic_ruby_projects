Bubble Sort
A simple Ruby implementation of the bubble sort algorithm created for The Odin Project.

What It Does
Bubble sort compares numbers next to each other and swaps them if they are out of order. The largest numbers slowly “bubble” to the end of the array after each pass.

Example:

bubble_sort([4,3,78,2,0,2])
# => [0,2,2,3,4,78]
